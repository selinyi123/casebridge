# v0.1.10 Assessment Schema and Outcome Tracking Checklist

## Expected API routes

- `GET /api/v1/cases/CASE-0001/outcomes`
- `POST /api/v1/cases/CASE-0001/outcomes`

## Manual flow

1. Open `/clients/c-0001/outcomes`.
2. Add a manual outcome narrative.
3. Select GAS score from -2 to +2.
4. Save outcome.
5. Confirm it appears in the outcome list.

## Acceptance rules

- GAS score must be between -2 and +2.
- Outcome creation is manual.
- AI does not score outcomes.
- Outcome can optionally reference goal or assessment.
- Outcome creation writes audit event.
