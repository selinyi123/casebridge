# v0.1.13 Service Plan Evidence Chain Plan

## Goal

Create a formal manual evidence chain from service plan to intervention to outcome.

## Scope

- service plan object
- intervention record object
- manual evidence chain endpoint
- evidence chain UI
- repository-level test

## Non-goals

- No AI-generated service plan
- No AI intervention scoring
- No automatic case closure
- No real external AI provider

## Acceptance

- Social worker can create a service plan manually.
- Social worker can create an intervention linked to a plan.
- Evidence chain returns plans, interventions, and outcomes.
- Evidence chain is manual-only.
- All write operations remain authenticated.
