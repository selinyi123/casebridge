# Issue Draft — v0.1.4 manual referral and goal gate

## Goal

Complete the manual case-service loop before AI integration.

## Scope

- service-goal API
- service-goal UI
- referral API
- referral UI
- agreement-state gate
- timeline-ready event shape for notes/goals/referrals

## Acceptance

- user can create a manual service goal
- user can create a manual referral from an existing resource
- referral cannot move to referred/success/completed without explicit agreement state
- resource matching stays candidate-only
- no AI provider is called
